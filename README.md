# Практична робота №8 — Streams in Node.js

## Мета роботи

Навчитися працювати зі streams у Node.js:

- readable streams
- writable streams
- transform streams
- підрахунок chunks та bytes
- обробка помилок stream

---

## Виконані завдання

- 8.1 STREAM FILE TO RESPONSE
- 8.2 STREAM UPLOAD
- 8.3 STREAM TRANSFORM
- 8.4 STREAM COUNTER
- 8.5 STREAM ERRORS

---

## Файли проєкту

- stream_response.js
- stream_upload.js
- stream_transform.js
- stream_counter.js
- stream_errors.js

---

## Запуск

Приклад запуску сервера:

```bash
node stream_response.js 3000
```
